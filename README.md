## 🚀 Featured Project — SecureFlow

<div align="center">

<a href="https://github.com/abhienix/SecureFlow">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=abhienix&repo=SecureFlow&theme=chartreuse-dark&border_color=00ff41&title_color=00ff41&icon_color=00ff41&text_color=7fff7f&bg_color=0d1117"/>
</a>

</div>

<br/>

```
SecureFlow — A 16-step DevSecOps pipeline I built from scratch

  Every push to main/dev triggers:

  [1] Secret Scan      → Gitleaks (blocks on leaked creds)
  [2] SAST             → Semgrep (OWASP Top 10 rule set)
  [3] Docker Build     → Containerize backend + frontend
  [4] CVE Scan         → Trivy (blocks on CRITICAL/HIGH)
  [5] Policy Gate      → Pass/fail decision point
  [6] Deploy           → Google Cloud Run
  [7] Observe          → Live Grafana + Prometheus dashboard

  Stack: Python · PostgreSQL · Redis · Docker · GitHub Actions · GCP
  Smart: Only rebuilds when relevant files actually changed
  Goal:  Security-first CI/CD — not an afterthought
```
