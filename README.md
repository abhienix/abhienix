### 🛡️ What I build

I'm passionate about **DevSecOps** — shifting security left in the development lifecycle so vulnerabilities are caught early, not in production.

- 🔍 Automated vulnerability scanning pipelines
- 🚦 Policy-based security enforcement
- 📊 Security observability with Prometheus & Grafana
- 🐳 Containerised, hardened deployment stacks

### 🚀 Featured Project

#### [SecureFlow](https://github.com/abhienix/SecureFlow) — DevSecOps Scanning Pipeline
> Trivy-powered container scanning with policy enforcement, Slack alerting, PostgreSQL scan history, and a live Grafana dashboard. One `docker compose up` and you're securing your stack.
