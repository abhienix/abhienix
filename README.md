## 🚀 Featured Project

<a href="https://github.com/abhienix/SecureFlow">
  <img align="center" src="https://github-readme-stats.vercel.app/api/pin/?username=abhienix&repo=SecureFlow&theme=chartreuse-dark&border_color=00ff41&title_color=00ff41&icon_color=00ff41" />
</a>

<br/><br/>

> **SecureFlow** — A DevSecOps pipeline that runs Trivy scans on Docker containers, enforces security policies (CRITICAL/HIGH block, MEDIUM warn), stores scan history in PostgreSQL, caches with Redis, alerts via Slack, and visualises everything in Grafana. One `docker compose up` and your stack is secured.
